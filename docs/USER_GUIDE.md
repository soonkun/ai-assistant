## 1. 대화하기

새싹이는 마이크로 말하거나 텍스트로 입력해 대화할 수 있습니다.  
완전 오프라인으로 동작하므로 민감한 내용도 외부로 유출되지 않습니다.

### 음성 대화
마이크에 대고 말하면 자동으로 인식합니다.  
말이 끝나면 자동으로 감지해 응답을 시작합니다.

### 텍스트 입력
화면 하단 입력창에 직접 타이핑해도 됩니다.

### 대화 예시

```
나:    오늘 오후 일정 알려줘
새싹이: 오늘 오후 3시에 팀 회의가 있어요. 그 외에는 일정이 없습니다!
```

```
나:    예산 승인 절차가 어떻게 돼?
새싹이: 내부 문서에서 찾았습니다. 예산 승인은 팀장 결재 후 기획팀에 제출하는 순서로 진행됩니다.
       (출처: 예산지침.pdf 15페이지)
```

## 2. 문서 검색

미리 등록해둔 내부 문서에서 필요한 정보를 찾아드립니다.  
PDF, Word, 텍스트 파일을 지원합니다.

### 사용 방법
문서를 `data/documents/` 폴더에 넣어두면 자동으로 학습합니다.  
이후 자연어로 질문하면 관련 내용을 찾아 답변합니다.

### 예시

| 질문 | 새싹이가 하는 일 |
|---|---|
| "출장비 신청 방법 알려줘" | 관련 규정 문서에서 해당 내용 검색 후 요약 |
| "지난달 보고서에 나온 수치 찾아줘" | 보고서에서 해당 내용 발췌 + 출처(파일명·페이지) 안내 |
| "스마트팜 사업 관련 자료 있어?" | 관련 문서 목록과 핵심 내용 정리 |

> **참고**: 문서를 처음 등록할 때 잠시 인덱싱 시간이 필요합니다 (새싹이가 `study` 표정으로 바뀝니다).

## 3. 일정 관리

"일정 추가해줘", "오늘 뭐 있어?" 같은 자연어로 일정을 관리합니다.

### 일정 추가

```
나:    내일 오전 10시에 예산 보고 회의 1시간 잡아줘
새싹이: 내일(4월 24일) 오전 10:00 ~ 11:00 예산 보고 회의를 등록했어요!
```

```
나:    5월 2일 오후 2시에 현장 방문 일정 추가해줘
새싹이: 5월 2일 오후 2:00 현장 방문 일정을 추가했습니다.
```

### 일정 조회

```
나:    이번 주 일정 알려줘
새싹이: 이번 주 일정입니다.
       - 4월 24일(목) 10:00  예산 보고 회의
       - 4월 25일(금) 14:00  팀 워크숍
```

## 4. 화면 분석

"지금 화면 봐줘"라고 하면 현재 화면을 캡처해 분석합니다.  
문서, 표, 오류 메시지 등 화면에 보이는 것은 무엇이든 질문할 수 있습니다.

### 예시

```
나:    지금 이 오류 메시지 뭐가 문제야?
새싹이: [화면 캡처 후] 오류 코드를 확인했어요. 파일 경로에 한글이 포함돼 있어서
       발생하는 문제입니다. 경로를 영문으로 변경하면 해결됩니다.
```

```
나:    이 엑셀 표에서 합계 맞는지 확인해줘
새싹이: [화면 캡처 후] 3행의 합계가 실제 값과 다르게 표시되어 있습니다.
       850만 원이 아닌 950만 원이 맞습니다.
```

### 연속 화면 공유

"화면 계속 보면서 알려줘"라고 하면 주기적으로 화면을 확인하며 변화를 감지합니다.  
"화면 공유 그만해"라고 하면 중단됩니다.

## 5. 회의록 자동 작성

회의 녹취록을 붙여넣으면 **개조식 회의 결과 보고서(.hwpx)** 를 자동으로 작성해 드립니다.

### 사용 방법

```
나:    회의록 만들어줘
       [녹취록 내용을 붙여넣기]

새싹이: 회의록을 작성하고 있습니다... (writing 표정으로 전환)
       완성됐어요! 아래 링크에서 다운로드하세요.
       → http://127.0.0.1:12393/download/abc123
       (24시간 내 다운로드 가능)
```

### 출력 형식

| 구분 | 내용 |
|---|---|
| 파일 형식 | 한글(HWP) `.hwpx` |
| 페이지 | 1장 또는 2장 선택 가능 |
| 작성 양식 | 개조식 (○ 항목 / - 세부 / * 부연) |
| 구성 | 제목, 일시·장소, 참석자, 주요 내용, 결정 사항, 금후 계획 |

### 1장 회의록 구조 예시

```
○ 2026년 스마트팜 보급 확대 추진 계획 논의
  - 총 예산 50억 원, 전국 20개 시군 대상 시범 운영
    * 4월 말 공모 공고, 5월 중 대상지 선정 예정
○ 농업인 교육 프로그램 개편 방향 검토
  - 온라인 교육 비중 30%에서 50%로 상향 결정
```

> **참고**: 회의록 생성에는 LLM 처리 시간이 필요합니다 (1장 기준 약 30초~2분).  
> 작성 중에는 새싹이가 `writing` 표정으로 바뀝니다.

## 알아두세요

- 새싹이는 **완전 오프라인**으로 동작합니다. 인터넷 연결이 없어도 사용할 수 있으며, 모든 데이터는 사내 PC에만 저장됩니다.
- 음성 인식은 **한국어**에 최적화되어 있습니다.
- 생성된 회의록 파일은 **24시간 후 자동 삭제**됩니다. 필요한 파일은 미리 저장해두세요.
- 화면 캡처 기능 사용 시 **개인정보가 포함된 화면**이 분석될 수 있으니 주의하세요.

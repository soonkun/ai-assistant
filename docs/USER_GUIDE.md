## 1. 앱 화면 구성

새싹이 앱은 **캐릭터 위젯**과 **패널** 두 부분으로 구성됩니다.

```
┌──────────────────────────────────────────────┐
│  [채팅] [캘린더] [문서] [회의록] [설정]   ✕  │  ← 탭 전환 (패널)
├──────────────────────────────────────────────┤
│                                              │
│         탭 내용 영역                         │
│                                              │
└──────────────────────────────────────────────┘

                 🌱  ← 새싹이 캐릭터 (클릭하면 패널 토글)
```

- **새싹이 클릭** → 패널 열기/닫기
- **패널 닫기(✕)** → 패널만 숨김. 캐릭터는 계속 화면에 표시
- **탭 전환** → 현재 작업 내용이 유지됩니다 (회의록 작업 중 다른 탭으로 갔다 와도 내용 보존)

## 2. 대화하기

새싹이는 마이크로 말하거나 텍스트로 입력해 대화할 수 있습니다.  
완전 오프라인으로 동작하므로 민감한 내용도 외부로 유출되지 않습니다.

### 음성 대화
마이크에 대고 말하면 자동으로 음성을 인식합니다.  
말이 끝나면 자동으로 감지해 응답을 시작합니다 (VAD 기반).

### 텍스트 입력
채팅 탭 하단 입력창에 직접 타이핑해도 됩니다.

### 대화 예시

```
나:     오늘 오후 일정 알려줘
새싹이:  오늘 오후 3시에 팀 회의가 있어요. 그 외에는 일정이 없습니다!
```

```
나:     예산 승인 절차가 어떻게 돼?
새싹이:  내부 문서에서 찾았습니다. 예산 승인은 팀장 결재 후 기획팀에 제출하는
         순서로 진행됩니다. (출처: 예산지침.pdf 15페이지)
```

## 3. 문서 검색 (RAG)

미리 등록해둔 내부 문서에서 필요한 정보를 찾아드립니다.

### 지원 파일 형식

| 형식 | 확장자 |
|---|---|
| PDF | .pdf |
| 워드 | .docx |
| 파워포인트 | .pptx |
| 한글 | .hwpx, .hwp |
| 텍스트 | .txt, .md |

### 문서 등록 방법

**문서 탭**에서 파일을 선택하거나, `data/documents/` 폴더에 직접 넣어두면 자동으로 인덱싱합니다.

> 처음 등록할 때 잠시 인덱싱 시간이 필요합니다. 이때 새싹이가 `study` 표정으로 바뀝니다.

### 예시

| 질문 | 새싹이가 하는 일 |
|---|---|
| "출장비 신청 방법 알려줘" | 관련 규정 문서에서 해당 내용 검색 후 요약 |
| "지난달 보고서에 나온 수치 찾아줘" | 보고서에서 해당 내용 발췌 + 출처(파일명·페이지) 안내 |
| "스마트팜 사업 관련 자료 있어?" | 관련 문서 목록과 핵심 내용 정리 |

## 4. 일정 관리

자연어로 일정을 관리합니다.

### 일정 추가

```
나:     내일 오전 10시에 예산 보고 회의 1시간 잡아줘
새싹이:  내일(4월 24일) 오전 10:00 ~ 11:00 예산 보고 회의를 등록했어요!
```

### 일정 조회

```
나:     이번 주 일정 알려줘
새싹이:  이번 주 일정입니다.
         - 4월 24일(목) 10:00  예산 보고 회의
         - 4월 25일(금) 14:00  팀 워크숍
```

### 자동 알림

등록된 일정 **10분 전**에 새싹이가 자동으로 음성 알림을 드립니다.  
매일 아침 **09:00**에는 오늘의 일정을 브리핑합니다.

## 5. 화면 분석

"지금 화면 봐줘"라고 하면 현재 화면을 캡처해 분석합니다.  
문서, 표, 오류 메시지 등 화면에 보이는 것은 무엇이든 질문할 수 있습니다.

### 예시

```
나:     지금 이 오류 메시지 뭐가 문제야?
새싹이:  [화면 캡처 후] 파일 경로에 한글이 포함돼 있어서 발생하는 문제입니다.
         경로를 영문으로 변경하면 해결됩니다.
```

```
나:     이 엑셀 표에서 합계 맞는지 확인해줘
새싹이:  [화면 캡처 후] 3행의 합계가 실제 값과 다릅니다.
         850만 원이 아닌 950만 원이 맞습니다.
```

## 6. 회의록 자동 작성

회의 녹음 파일을 업로드하면 **개조식 회의 결과 보고서(.hwpx)** 를 자동으로 작성해 드립니다.  
**회의록 탭**에서 3단계로 진행됩니다.

---

### Step 1 — 전사 (녹음 → 텍스트)

**지원 오디오 형식**: WAV, MP3, M4A, OGG, FLAC, WEBM 등

1. "오디오 파일 선택" 버튼을 눌러 회의 녹음 파일을 선택합니다.
2. "전사 시작" 버튼을 클릭합니다.
3. Whisper AI가 음성을 텍스트로 변환합니다 (45분 녹음 기준 약 3~10분 소요).
4. 전사가 완료되면 새싹이가 음성으로 알려줍니다:
   > *"전사가 완료되었어요. 확인해 보시고 회의록 작성을 시작해주세요."*
5. 전사 결과 텍스트를 확인·수정하고, "전사본 다운로드(.txt)" 버튼으로 저장할 수 있습니다.

---

### Step 2 — 회의록 작성 (텍스트 → 회의록)

1. Step 1의 전사 결과가 자동으로 입력됩니다 (직접 수정하거나 다른 텍스트를 붙여넣을 수도 있습니다).
2. **분량 선택**: 1페이지 또는 2페이지 중 선택합니다.
   - 1페이지: 핵심 논의 사항과 결정 사항 위주로 간결하게
   - 2페이지: 논의 흐름, 쟁점, 근거, 세부 결정 사항까지 상세히
3. "회의록 작성" 버튼을 클릭합니다.
4. LLM이 개조식 회의록을 작성합니다 (약 30초~2분 소요).
5. 완료되면 새싹이가 음성으로 알려줍니다:
   > *"회의록 작성이 완료되었습니다. 확인해보시고 결과보고서 작성을 시작하세요."*
6. 회의록 내용을 확인·수정하고, "회의록 다운로드(.txt)" 버튼으로 저장할 수 있습니다.

> **팁**: 분량 설정은 여기서 결정됩니다. 2페이지짜리 결과보고서를 원하신다면 **반드시 2페이지를 선택**하세요. 회의록 자체가 1페이지 분량이면 결과보고서도 1페이지 수준으로만 생성됩니다.

---

### Step 3 — 결과보고서 생성 (.hwpx)

1. Step 2의 회의록이 자동으로 입력됩니다 (직접 수정도 가능합니다).
2. "결과보고서 생성" 버튼을 클릭합니다.
3. 한글(.hwpx) 문서를 생성합니다 (약 10~30초 소요).
4. 완료되면 새싹이가 음성으로 알려줍니다:
   > *"결과보고서를 완성하였습니다. 내용을 검토하세요."*
5. "결과보고서 다운로드(.hwpx)" 버튼을 클릭해 저장합니다.

---

### 출력 결과보고서 형식

| 구분 | 내용 |
|---|---|
| 파일 형식 | 한글(HWP) `.hwpx` |
| 작성 양식 | 개조식 (○ 항목 / - 세부 / * 부연) |
| 구성 항목 | 제목, 일시·장소, 참석자, 주요 내용, 결정 사항, 향후 계획 |
| 날짜 처리 | 녹취록에 날짜가 없으면 오늘 날짜 자동 입력 |

### 1페이지 회의록 구조 예시

```
○ 2026년 스마트팜 보급 확대 추진 계획 논의
  - 총 예산 50억 원, 전국 20개 시군 대상 시범 운영
    * 4월 말 공모 공고, 5월 중 대상지 선정 예정
○ 농업인 교육 프로그램 개편 방향 검토
  - 온라인 교육 비중 30%에서 50%로 상향 결정
```

## 7. 펫 모드

**설정 탭**에서 "펫 모드" 를 활성화하면 새싹이가 바탕화면 위에 항상 떠 있는 작은 위젯으로 바뀝니다.

| 기능 | 동작 |
|---|---|
| **항상 위** | 다른 앱 창이 열려도 새싹이가 항상 위에 표시 |
| **클릭 관통** | 새싹이 뒤에 있는 앱을 그대로 클릭할 수 있음 |
| **드래그 이동** | 새싹이를 클릭·드래그해 원하는 위치로 이동 |
| **위치 저장** | 앱을 재시작해도 마지막 위치에서 시작 |

> **참고**: 펫 모드에서도 새싹이를 클릭하면 패널이 열립니다.

## 9. 알아두세요

- 새싹이는 **완전 오프라인**으로 동작합니다. 인터넷 연결이 없어도 사용할 수 있으며, 모든 데이터는 사내 PC에만 저장됩니다.
- 음성 인식은 **한국어**에 최적화되어 있습니다.
- 생성된 회의록 파일은 **24시간 후 자동 삭제**됩니다. 필요한 파일은 미리 저장해두세요.
- 화면 캡처 기능 사용 시 **개인정보가 포함된 화면**이 분석될 수 있으니 주의하세요.
- **탭 전환이나 패널 닫기**를 해도 회의록 작업 내용이 유지됩니다. 안심하고 다른 탭을 이용하세요.

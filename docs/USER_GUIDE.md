# 새싹이 사용 가이드

> 새싹이는 인터넷 없이 **내 PC에서 동작하는 사내 AI 비서**입니다. 묻고 답받고, 사내 문서에서 근거를 찾고, 업무 노트·일정·회의 결과보고서를 자동으로 만들어 줍니다.
>
> 이 문서는 "어떻게 쓰는가"에 초점을 둔 실용 가이드입니다. 처음이라면 **1장 → 필요한 작업(3~6장)** 순서로 보세요.

---

## 새싹이로 할 수 있는 일 (한눈에)

| 하고 싶은 것 | 이렇게 하세요 | 자세히 |
|---|---|---|
| 사내 규정·보고서 내용 물어보기 | 문서를 등록해두고 그냥 질문 | [3장](#3-사내-문서에서-답-찾기) |
| 오늘 한 일을 노트로 정리 | "오늘 ~~ 했어" + (자료 첨부) | [4장](#4-업무-노트-만들기) |
| 화면 캡처/한글파일 내용 정리 | 스크린샷 붙여넣기 또는 파일 첨부 후 "정리해줘" | [4장](#4-업무-노트-만들기) |
| 일정 등록·확인 | "내일 2시 회의 잡아줘" / "이번 주 일정 뭐 있어?" | [5장](#5-일정-관리) |
| 회의 녹음 → 한글 결과보고서 | 회의록 탭에서 3단계(전사→회의록→보고서) | [6장](#6-회의록-만들기-한글-결과보고서) |
| 손 안 대고 음성으로 대화 | 마이크 버튼 누르고 말하기 | [2장](#2-새싹이와-대화하기) |

## 1. 시작하기

### 실행

- 바탕화면의 **새싹이** 아이콘(또는 `새싹이.cmd`)을 실행합니다.
- 처음 켜질 때 잠깐 준비 시간이 필요합니다(AI 서버 기동). 준비가 끝나면 새싹이가 인사를 건넵니다.

### 두 가지 모드

| 모드 | 모습 | 언제 쓰나 |
|---|---|---|
| **창 모드** | 일반 데스크톱 창. 위쪽에 탭(새싹이·일정표·노트·문서·회의록·설정) | 문서 등록, 노트 편집, 회의록 작성 등 본격 작업 |
| **펫 모드** | 바탕화면에 떠 있는 작은 새싹이 캐릭터 | 평소 옆에 두고 빠르게 질문할 때 |

- **모드 전환**: 설정 탭에서 펫 모드를 켜고 끕니다.
- 펫 모드에서는 캐릭터를 **드래그**해 옮길 수 있고, 캐릭터를 클릭하면 대화창이 열립니다. 캐릭터 바깥 영역 클릭은 바탕화면으로 그대로 전달됩니다(작업에 방해되지 않음).

## 2. 새싹이와 대화하기

대화창 아래 입력줄에서 세 가지 방법으로 말을 겁니다.

### 텍스트
입력창에 쓰고 Enter. (한글 조합 중 Enter는 무시되니 안심하고 입력하세요.)

### 음성
**마이크 버튼**을 누르고 말하면 음성을 글로 받아 적어 질문합니다(말이 끝나면 잠시 후 자동 종료). 인터넷 없이 동작합니다.

### 이미지 / 스크린샷
- 화면을 캡처해 **그대로 붙여넣기(Ctrl/⌘+V)** 하거나 파일로 첨부합니다.
- 새싹이가 **이미지 속 글자·표·수치·날짜·담당자를 읽어** 답하거나 노트로 정리합니다. (한글 스크린샷도 인식)

### 답변 중 표시
새싹이가 일하는 동안 캐릭터 위에 현재 단계가 뜹니다: *"질문을 살펴보고 있어요…" → "자료를 찾아보고 있어요…" → "답변을 작성하고 있어요…"*. 노트를 저장할 땐 노트 쓰는 표정으로 바뀝니다.

> 💡 **새싹이는 입력을 먼저 분류합니다.** "일정인지 / 문서 질문인지 / 노트로 저장할지 / 그냥 대화인지"를 스스로 판단해 알맞게 처리합니다. 그래서 특별한 명령어 없이 자연스럽게 말해도 됩니다.

## 3. 사내 문서에서 답 찾기

미리 등록해 둔 문서에서 **근거(출처)와 함께** 답을 찾아 줍니다.

### 3‑1. 먼저 문서를 등록 (문서 탭)

1. **문서 탭**으로 이동
2. (선택) **새 폴더**로 주제별 폴더를 만듭니다 — 예: `규정·지침`, `회의 자료`, `업무편람`
3. 파일을 등록합니다:
   - **드래그 앤 드롭** — 탐색기에서 문서 탭 위로 끌어다 놓기, 또는
   - **클릭하여 파일 선택**
   - 대화창에 파일을 첨부해도 자동 등록됩니다
4. 등록 중에는 새싹이가 *책을 책장에 넣는* 모습으로 바뀌고, 끝나면 검색 가능해집니다.

지원 형식: **PDF · 워드(docx) · 파워포인트(pptx) · 한글(hwpx) · 텍스트(txt/md)**

### 3‑2. 그냥 질문하기

문서 탭에 들어갈 필요 없이, 새싹이 대화창에서 평소처럼 물어보면 됩니다.

```
나:     출장비 신청 절차 알려줘
새싹이:  출장 후 5일 이내 지출결의서를 작성해 총무팀에 제출하면 됩니다.
         (출처: 출장비지급규정.hwpx)   ← 클릭하면 원본이 바로 열립니다
```

- 답변에는 **근거가 된 문서 출처**가 칩으로 붙습니다. 칩을 클릭하면 원본 파일이 **저장 위치를 묻지 않고 기본 앱(한글·PDF 뷰어 등)으로 바로 열립니다.**
- 문서에서 못 찾으면 "등록된 문서에서 찾지 못했어요"라고 솔직히 알려 줍니다(지어내지 않음).

### 3‑3. 문서 정리

- 문서 옆 🗑️ 로 문서를 삭제(검색 인덱스도 함께 제거).
- 폴더 삭제 시 안의 문서를 함께 지울지 선택할 수 있습니다.

## 4. 업무 노트 만들기

처리한 업무·회의·자료를 **업무 노트**로 정리해 모아 둡니다. (노트 탭에서 보고 편집)

### 4‑1. 가장 쉬운 방법 — 대화로 시키기

자료와 함께 한 줄만 적으면, 새싹이가 **자료의 실제 내용을 근거로** 제목·요약이 들어간 노트를 만들어 줍니다.

| 상황 | 입력 예 |
|---|---|
| 한글/PDF 파일을 정리 | 파일 첨부 + `"오늘 이 회의 진행했어"` |
| 화면 캡처를 정리 | 스크린샷 붙여넣기 + `"이 내용 노트로 정리해줘"` |
| 그냥 구두 보고 | `"오늘 출장비 정산 처리했어"` |

- 노트가 만들어지면 새싹이가 **노트 작성 모드**로 바뀌고 "업무 노트를 작성할게요!"라고 안내한 뒤 저장합니다.
- 노트는 **상황 / 절차 / 사용 자료 / 교훈** 구조로 정리되고, 첨부 자료의 수치·날짜·담당자가 보존됩니다.
- 노트 안의 **관련 자료**를 클릭하면 원본이 기본 앱으로 바로 열립니다.

### 4‑2. 직접 작성

노트 탭의 **+ 버튼**으로 빈 노트를 만들어 직접 씁니다. 창 모드에서는 Notion 스타일 블록 에디터(제목·목록·표 등)를 지원합니다.

### 4‑3. 노트 그래프

노트 탭에서 노트들 사이의 **연결 관계를 그래프**로 볼 수 있습니다.

> 업무 노트는 마크다운 파일로 **내 PC에 저장**됩니다.

## 5. 일정 관리

### 등록·조회

말로 하면 됩니다.

```
나:     다음 주 화요일 오후 3시 팀 회의 잡아줘
새싹이:  '팀 회의'를 화요일 오후 3시로 등록했어요.

나:     이번 주 일정 뭐 있어?
새싹이:  (현재 등록된 일정을 조회해 알려줍니다)
```

- 일정표 탭에서 달력으로 보고 직접 추가/수정/삭제할 수도 있습니다.

### 알아서 챙겨주는 알림

- **아침 브리핑** (매일 09:00): 그날 일정을 정리해 알려줍니다.
- **일정 임박 알림**: 일정 시작 전 음성으로 알려줍니다.
- **휴식 권고**: 너무 오래 쉬지 않고 일하면 잠깐 쉬라고 권합니다.

## 6. 회의록 만들기 (한글 결과보고서)

**회의록 탭**은 다른 기능과 독립적으로 동작하는 3단계 파이프라인입니다. 회의 녹음을 사내 공문서 양식의 **한글(.hwpx) 결과보고서**까지 만들어 줍니다.

```
Step 1  전사        녹음 파일 → 글(녹취 텍스트)
Step 2  회의록 작성  녹취 → 개조식 회의록 (○ / - / * 위계, [개요]·[주요내용]·[향후계획])
Step 3  결과보고서    회의록 → 한글(.hwpx) 공문서로 다운로드
```

### 사용 순서

1. **Step 1 — 전사**: 회의 녹음 파일을 올리면(드래그도 가능) 한국어로 받아 적습니다. 이미 정리된 텍스트가 있으면 이 단계를 건너뛰고 바로 붙여넣어도 됩니다.
2. **Step 2 — 회의록 작성**: **1장 / 2장** 분량을 고르고 "생성"을 누르면, 사람이 읽고 수정할 수 있는 **개조식 회의록**이 만들어집니다. 여기서 내용을 직접 다듬을 수 있습니다.
3. **Step 3 — 결과보고서 생성**: 다듬은 회의록을 **한글(.hwpx) 공문서**로 변환해 다운로드합니다.

### 결과보고서 형식

- 제목 / [개요](일시·장소·참석자·목적) / [주요내용](○-* 위계) / [향후계획]
- 공무원 개조식 문체(명사형 종결 우선), 녹취의 수치·금액·날짜·기관명 보존.

> 💡 분량이나 문체가 마음에 안 들면 Step 2 결과를 직접 고친 뒤 Step 3로 넘기면 됩니다.

## 7. 화면(탭)별 빠른 참조

| 탭 | 하는 일 |
|---|---|
| **새싹이** | 대화(텍스트·음성·이미지), 질문·노트·일정 등 대부분의 요청 |
| **일정표** | 달력으로 일정 보기·추가·수정·삭제 |
| **노트** | 업무 노트 보기·편집, 노트 그래프 |
| **문서** | 사내 문서 등록(폴더 관리)·삭제, RAG 검색 대상 관리 |
| **회의록** | 전사 → 회의록 → 한글 결과보고서(독립 기능) |
| **설정** | 모델·음성·화면·지침 설정 |

## 8. 설정

설정 탭에서 동작 방식을 바꿉니다.

- **화면**: 다크/라이트 테마, 글씨 크기(창 모드·펫 모드 각각 저장), 펫 모드 전환.
- **AI 모델(LLM)**: 로컬(Ollama) ↔ OpenAI 전환, 모델 선택.
  > 현재 OpenAI는 사내 인트라넷 GPU가 갖춰지기 전까지의 임시 대역입니다. 인트라넷 모델로 바꾸면 외부 호출 없이 동작합니다.
- **음성(TTS)**: 시스템 음성 / MeloTTS(완전 오프라인) 중 선택, 재생 속도 조절, "목소리 테스트".
- **의도 분류기 / 지침(고급)**: 입력 자동 분류 on/off, 새싹이의 행동 지침(페르소나·노트/문서 답변 가이드 등)을 직접 편집·기본값 복원.

> **이미지 인식이 잘 안 될 때**: 이미지(OCR)는 별도의 비전 모델을 씁니다. 관리자라면 설정 파일 `conf.yaml`의 `app.ollama.vision_model`(예: `qwen2.5vl:7b`)을 확인하세요. 해당 모델은 미리 `ollama pull`로 받아 두어야 합니다.

## 9. 자주 묻는 질문 & 문제 해결

**Q. 인터넷이 안 되는데 써도 되나요?**
네. 모든 기능이 내 PC에서 동작하도록 설계되어 있습니다. (단, 설정에서 OpenAI를 선택한 경우에만 외부 호출이 발생합니다.)

**Q. 첨부했는데 노트에 내용이 안 담겨요 / "정리할게요"만 하고 안 만들어요.**
새싹이를 **완전히 종료 후 재실행**해 보세요(최신 동작 적용). 그래도 그러면, 그때 화면에 나온 답변과 함께 알려주시면 원인을 잡습니다.

**Q. 출처/관련 자료를 클릭하면 다운로드 창이 떠요.**
최신 버전에서는 **바로 열림**으로 바뀌었습니다. 재실행 후에도 다운로드 창이 뜨면 알려주세요.

**Q. 앱이 켜지다가 멈춰요 / 안 떠요.**
AI 서버(Ollama)나 백엔드 준비 중일 수 있습니다. 검은 콘솔 창에 메시지가 남으니, 마지막 메시지를 확인해 주세요. 보통은 잠시 기다리거나 재실행하면 해결됩니다.

**Q. 캐릭터가 바탕화면 클릭을 가로채요.**
펫 모드에서 캐릭터 바깥은 클릭이 통과하도록 되어 있습니다. 방해되면 설정에서 펫 모드를 끄거나 캐릭터를 구석으로 옮기세요.

**Q. 문서에서 답을 못 찾아요.**
문서 탭에 해당 파일이 등록(인덱싱)되어 있는지 확인하세요. 등록 직후에는 검색에 반영됩니다. 같은 파일을 다시 올리면 덮어씁니다(중복 없음).

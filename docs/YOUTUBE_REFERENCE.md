# 참조 — 유튜브 조사 요약

이 문서는 프로젝트 착수 당시 사용자가 참고한 유튜브 두 영상의 내용 요약이다. Planner·Researcher가 Open-LLM-VTuber 기반 구축 시 맥락 참고용.

## 1. AI 아바타 시장 동향

- 과거 사람이 모델을 조작하던 방식에서 AI가 스스로 판단·행동하는 AI 버튜버 시대로 전환.
- 시조 사례: **Neuro-sama** — 리듬 게임 AI에서 시작, 현재는 합방·토크쇼·AI끼리 대화(Evil Neuro)까지.
- 오픈소스·API 접근성 향상으로 진입장벽 급락.
- 로컬 LLM(프라이버시·비용·오프라인)이 핵심 트렌드.

## 2. 구체 구현 방법 (영상의 요점)

### 구성요소
뇌(LLM) + 시각(화면) + 청각(STT) + 목소리(TTS) + 신체(Live2D/3D).

### 주요 오픈소스
- **AIRI** — 마인크래프트·팩토리오 등 게임 강제 플레이 기능 포함.
- **Open LLM VTuber** — 안정성 강점, 감정 기반 페이셜 애니, 로컬 AI 연결 쉬움. **본 프로젝트의 베이스.**

### 설치 단계 (영상 기준)
1. Python 설치 + PATH 등록.
2. VS Code 준비.
3. Open LLM VTuber 릴리즈에서 zip 다운로드·압축해제.
4. UV 라이브러리로 빠른 설치.
5. ffmpeg 설치 (음성 인식 필수).
6. `uv sync` 로 의존성 자동 설치.

### 설정 (conf.yaml)
- LLM 공급자: OpenAI API 또는 Ollama (로컬).
- 모델: 비용 고려해 gpt-4o-mini 등 또는 로컬.
- TTS: Edge TTS 기본, 한국어 여성 보이스 지원. ElevenLabs·로컬 보이스 카피 가능.

### 확장
- **MCP(Model Context Protocol)**: AI에 도구 사용 능력 부여.
  - DuckDuckGo 검색 (실시간 정보).
  - GitHub MCP로 PR 검토.
- **화면·얼굴 인식**: 웹캠·스크린 캡처로 상호작용.
- **감정 표현**: 텍스트 괄호 안 감정 데이터를 Live2D 표정에 매핑.

## 3. 로컬 LLM·모바일 동향

### Gemma
- 작은 모델이 이전 세대 27B급 성능 압도.
- 상업 라이선스 가능.
- "구글 AI 엣지 갤러리" 앱으로 Android/iOS에서 오프라인 실행.

### 도구
- **LM Studio**: 로컬 모델 다운로드·채팅·서버 가동.
- **Ollama**: 로컬 LLM 실행·관리 최적화. 본 프로젝트 기본 선택.
- **OpenCode**: 에이전트 기능, 로컬 서버 연결해 문서 요약·코딩.

## 4. 로컬 AI 장단점

### 장점
- 데이터 보안 (오프라인).
- API 비용 없음.
- 개인화·튜닝 자유.

### 단점
- 복잡·긴 코드 생성 능력 떨어짐.
- 기본 응답이 영어인 경우 많음 → 시스템 프롬프트 튜닝 필요.
- VRAM 의존도 높음.

## 5. 본 프로젝트에서 이 요약을 어떻게 쓰는가

- **베이스 프로젝트**: Open-LLM-VTuber로 확정. AIRI는 게임 자동화 성격이라 업무 비서 용도와 안 맞음.
- **LLM**: Gemma 4 E4B + Ollama. 영상은 Gemma 3 언급이지만 2026년 4월 현재 Gemma 4가 나왔고 E4B 등급이 오프라인 멀티모달에 맞다.
- **TTS**: 영상의 Edge TTS는 인터넷 필요 → **사내 인트라넷 요구사항 위반**. Piper(한국어) 또는 CosyVoice 로컬로 대체.
- **MCP**: 인트라넷이라 DuckDuckGo 불가. 대신 사내 위키·파일시스템 MCP.
- **감정 태그 프로토콜**: 영상의 "괄호 감정 데이터" 방식을 그대로 채택 (docs/CHARACTER_SAESSAGI.md 참조).

## 6. 영상 대비 본 프로젝트의 차이

| 영상 | 본 프로젝트 |
|---|---|
| 개인 PC, 가끔 API 호출 허용 | 완전 오프라인 강제 |
| Gemma 3 | Gemma 4 E4B |
| Edge TTS (온라인) | Piper/CosyVoice (오프라인) |
| Live2D 샘플 모델 | 새싹이(커스텀 PNG 스프라이트) |
| 게임·토크쇼 용도 | 업무 비서 (RAG·일정·유휴·화면 인식) |
| DuckDuckGo MCP | 사내 문서 RAG 주력 |

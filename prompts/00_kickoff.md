# 00 — Kickoff 프롬프트

Claude Code를 이 프로젝트 폴더에서 열고 처음 붙여넣을 프롬프트.

---

```
프로젝트 오리엔테이션을 시작한다.

1단계:
- README.md, REQUIREMENTS.md, CLAUDE.md를 처음부터 끝까지 읽어라.
- 이해한 내용을 짧게(10줄 이내) 요약해서 확인해 달라. "내가 이해한 바가 맞냐"고 나에게 확인을 요청해라.

2단계 (내가 1단계 확인을 준 뒤에만 진행):
- .claude/agents/ 의 모든 에이전트 정의를 읽어라.
- 어떤 역할이 어떤 모델을 쓰고 언제 호출되는지 표로 정리.
- "이 운영 규칙에 모호한 부분이 있는가?"를 스스로 검토하고, 있다면 질문으로 나에게 올려라.

3단계 (내가 2단계 확인을 준 뒤에만 진행):
- upstream/ 디렉토리가 비어 있는지 확인.
- 비어 있다면 내게 "scripts/bootstrap.sh (또는 bootstrap.ps1)를 먼저 실행해 Open-LLM-VTuber를 clone해 주세요"라고 안내.
- 이미 clone되어 있다면 Phase 0로 넘어갈 준비가 됐음을 알려라.

절대 지금 코드를 작성하지 마라. 절대 파일을 수정하지 마라. 너는 읽고 확인하는 단계다.
```

---

**기대 응답:** Claude가 "이해했습니다"와 함께 REQUIREMENTS 요약 10줄과 확인 질문을 올려야 한다. 만약 코드 생성이나 폴더 수정을 시도하면 멈추고 "지금은 Phase 0 이전 단계다"라고 제지할 것.
